# Indian_General_Election_2024_Analysis

📌 Overview

This project presents a comprehensive analysis of the India General Elections 2024 dataset using SQL and Power BI.

It focuses on extracting actionable insights such as:

-Seat distribution

-Party and alliance performance

-Candidate-level analysis

-Voting patterns (EVM vs Postal)


🎯 Objectives

-Analyze election results at state and constituency level<br>
-Compare performance of NDA, I.N.D.I.A, and OTHER alliances<br>
-Identify top-performing parties and candidates<br>
-Build an interactive dashboard for decision-making<br>
